Project Overview

This REU project integrates experimental submerged fermentation studies with data-driven machine learning models to analyze, predict, and optimize marine protease production. A marine-derived bacterial isolate was employed for extracellular protease production, and fermentation kinetics were experimentally monitored. Artificial Neural Network (ANN) and Random Forest models were developed to capture nonlinear relationships between fermentation parameters and enzyme activity.

Problem Statement & Motivation

Conventional optimization of enzyme production relies heavily on trial-and-error experimentation, which is time-consuming and resource intensive. Fermentation systems exhibit nonlinear behavior influenced by multiple interacting parameters, making prediction difficult using classical approaches alone.

Marine microorganisms offer robust enzymes with superior stability, but systematic modeling and optimization of marine protease production remain limited. This project aimed to bridge experimental fermentation studies with artificial intelligence–based modeling to improve prediction and support data-driven bioprocess optimization.

Methodology

Wet Lab Methodology

  • Isolation and screening of marine bacteria using skim milk agar for extracellular protease production
  • Selection of a high-performing marine isolate (VSD-27) based on clear zone formation
  • Submerged batch fermentation in shake flasks using skim milk–based production medium
  • Monitoring of fermentation over 120 hours under alkaline conditions
  • Biomass estimation using optical density (OD660)
  • Protease activity estimation using the Lowry–Folin colorimetric assay

Dry Lab / Computational Methodology

  • Compilation and preprocessing of fermentation data
  • Polynomial regression and classical fermentation kinetic analysis
  • Development of Artificial Neural Network (ANN) model for enzyme activity prediction
  • Implementation of Random Forest regression model
  • Model evaluation using R² and RMSE metrics
  • Feature importance analysis and Bayesian optimization framework

Results & Key Findings

  • The selected marine isolate (VSD-27) showed strong extracellular protease activity
  • Protease production was growth-associated, peaking during late exponential to early stationary phase
  • Fermentation kinetics revealed a strong temporal dependency of enzyme production
  • ANN model achieved an R² value of 0.9549 for enzyme activity prediction
  • Random Forest model outperformed ANN with an R² of 0.9820 and lower RMSE
  • Feature importance analysis identified fermentation time as the most influential parameter

The Random Forest model provided superior predictive performance, highlighting the value of ensemble learning approaches in bioprocess systems.
